Программа генерации случайных паролей.
Автор: Янин Ярослав Иванович, группа МК-101

Опции командной строки:
  -minl <число>   Минимальная длина пароля
  -maxl <число>   Максимальная длина пароля
  -n <число>      Точная длина пароля 
  -c <число>      Количество паролей (по умолчанию 1)
  -a <строка>     Пользовательский алфавит 
  -C <a|A|D|S>    Наборы символов (a - строчные, A - заглавные, D - цифры, S - спецсимволы)
  -d <символ>     Добавить разделитель значений (по умолчанию "=" и ":")
  -D <символ>     Заменить разделители
  -p <числа>      Вероятности выпадения символов/наборов через запятую (например: -p 0.5,0.2)

Примеры запуска:
> my_program.exe -n 12 -C D -c 3
> my_program.exe -minl 8 -maxl 15 -a abc!@# -c 5
> my_program.exe -n 50 -C aD -p 0.95

СБОРКА И ТЕСТИРОВАНИЕ ПРОЕКТА (CMake)

Подготовка и конфигурация:
> mkdir build
> cd build
> cmake -S .. -G "Visual Studio 17 2022" -A Win32

(Для динамической библиотеки добавить флаг: -DBUILD_SHARED_LIBS=ON)

Сборка:
> cmake --build . --config Release
> cmake --build . --config Debug

Запуск тестов (выполнять в папке build):
> cmake --build . --target test_all
> ctest -C Release
